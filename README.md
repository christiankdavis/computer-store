# Refurbished Computer Store

Small React + TypeScript app that lets you browse refurbished computers, filter by category, and add items to a cart.

## Run the project

1. Install dependencies
   `npm install`
2. Start the dev server
   `npm run dev`

3. Open the local URL shown in the terminal (usually something like `http://localhost:5173`).

## Notes

- Products are fetched from the provided JSON endpoint.
- Users can filter products by category.
- Items can be added to the cart and quantities can be increased or decreased.
- Cart totals update automatically.
